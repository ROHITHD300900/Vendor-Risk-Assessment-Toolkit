![Third-Party Risk](https://img.shields.io/badge/Focus-TPRM-blue.svg) ![Security](https://img.shields.io/badge/Security-Assessment-green.svg) ![Due Diligence](https://img.shields.io/badge/Process-Due_Diligence-orange.svg)

## 📋 Project Overview
The **Vendor Risk Assessment Toolkit** provides a standardized framework for evaluating the security posture of third-party service providers, SaaS vendors, and business partners. This toolkit ensures that organizations can identify, assess, and mitigate risks associated with their supply chain.

## 🚀 Business Value
*   **Risk Mitigation:** Identifies critical security gaps in vendor environments before data sharing occurs.
*   **Standardized Process:** Eliminates inconsistent assessment methodologies across different departments.
*   **Compliance Alignment:** Maps vendor assessments to ISO 27001 (A.15), SOC 2 (CC9.0), and GDPR (Article 28) requirements.

## 🛠️ Toolkit Components
1.  **Vendor Security Questionnaire (VSQ):** A 100+ point questionnaire covering Network Security, Data Protection, HR, and Compliance.
2.  **Risk Scoring Model:** An automated scoring matrix (Excel-based) to categorize vendors as High, Medium, or Low risk.
3.  **Due Diligence Checklist:** Essential documents to request from vendors (SOC 2 reports, Pen Test summaries, Insurance certs).
4.  **Vendor Risk Register:** A template for tracking the lifecycle of third-party risks.
